# 名言书摘类微头条创作提示词 - 完整版

## 核心目标
创作**450-550字**名言书摘类微头条（严格控制！）
- AI检测率：0%
- 完读率：80%+
- 评论率：5%+

### ⚠️ 字数红线（必须遵守）
- **最少**：400字
- **最佳**：450-550字
- **最多**：600字（绝对不能超过！）
- **超过600字立即停止，删减内容！**

---

## 第一部分：完读率三大核心机制（必须执行）

### 机制1：前3秒钩子 ★★★★★

**问题**：70%的人会在前3秒离开，必须立即抓住注意力

**解决方案**：金句后立即用"你"代入读者，或用强对比制造共鸣

#### 方法A：代入式开头（最推荐！参考对标作品）

**开头引用格式**（根据素材选择）：
- 有人名/书名：`杨绛先生说：` `鲁迅说：` `《XX》里有句话：`
- 无人名：从以下随机选择（不要每篇都一样！）
  - `一段很有道理的话：`
  - `一段醍醐灌顶的话：`
  - `看到一句话，说得很在理：`
  - `一句道尽世情真相的话：`

**⚠️ 过渡句要多样化！（不要每篇都用同一个比喻）**

❌ **禁止重复使用**：
- "这话，像一把冰冷的尺子..."（用过一次就换）
- "这话，像一面冰冷的镜子..."（用过一次就换）

✅ **过渡句池（随机选择）**：
- 这话太扎心了。
- 短短几句话，道破了XX的真相。
- 这话说到点子上了。
- 细想一下，还真是这么回事。
- 越琢磨越觉得有道理。
- 这话听着扎心，但的确是现实。
- 说白了，就是这么回事。
- （也可以不用过渡句，直接进入"你是不是也这样"）

**案例**（参考对标作品2）：
```
杨绛先生有一段话，一针见血：
"人控制钱，是生活；钱控制人，是生存。"

短短两句话，道破了金钱与人生关系的终极真相。

你是不是也这样？
不敢停下来，因为下个月的房租、孩子的学费、父母的药费，都像闹钟一样准时催着你。
```

#### 方法B：强对比开头

```
一段很有道理的话：
"【金句】"

同样是XX，有的人【场景A】，有的人【场景B】。
你说，这能一样吗？
```

#### 方法C：微故事开头（慎用！容易写太长）

```
一段很有道理的话：
"【金句】"

我一个朋友/我邻居...
【注意：故事必须控制在2-3句话内，不要展开细节！】
```

**⚠️ 重要警告**：
- **不要讲长故事！** 对标作品大多用"你"代入，而非讲具体朋友的故事
- 如果用故事，**最多2-3句**，然后立即转到"你"
- 故事是引子，不是主体
- **错误示范**：花200字讲朋友的经历 → 这会让文章变成"别人的故事"
- **正确示范**：用"你"让读者觉得"说的就是我"

---

### 机制2：对比反转结构（中段防流失）★★★★★

**问题**：读到中间容易疲劳，需要"意外"刺激

**解决方案**：用强烈对比制造冲击，而非讲长故事

#### 对比公式（推荐）：

```
同样是【某件事】。
有的人【场景A - 轻松/奢侈】；
有的人【场景B - 辛苦/窘迫】。
你说，这能一样吗？
```

**示例**（参考对标作品）：
```
同样是出一身汗。
有的人在健身房里跑步，出了汗，洗个澡，喝个蛋白粉，舒舒服服；
有的人在马路牙子上喘气，出了汗，喝口两块钱的矿泉水，接着往前骑。
你说，这能一样吗？

同样是为钱发愁。
有的人愁的是，这套500万的江景房要不要换个大点儿的；
有的人愁的是，这500块的水电费月底能不能交上。
```

#### 悬念公式（可选，注意简短）：

```
【设置常规预期，1句话】
【突然反转，1-2句话】
```

**常用反转点**：
- 以为有钱人也烦恼，结果烦恼的层次不同
- 以为大家都一样，结果差距天壤之别
- 以为是共同话题，结果根本不在一个频道

---

### 机制3：转折升华（2/3处插入）★★★★

**问题**：文章读到2/3会产生疲劳，容易弃读

**解决方案**：在倒数第2-3段插入"转折"或"希望"

#### 推荐方式：

**方式A：但话说回来（转折）**
```
但话说回来。
钱没了，可以再赚；情散了，可以再遇。
人这一辈子，没有谁不栽跟头的。
普通人想改变命运，确实不容易，可也不是一点儿希望都没有。
```

**方式B：俗语引入新角度**
```
老话说得好，夏虫不可语冰，井蛙不可语海。
不是一个世界的人，连烦恼都不在一个频道上。
```

**方式C：简短新案例（注意：不要展开！）**
```
前两天看到个新闻，有个企业家，企业倒闭了。
你看，有的人崩溃是从几百万开始的；有的人崩溃，可能就是一场感冒、一次失业。
```

**⚠️ 注意**：
- 不要在这里再讲一个长故事
- 如果用案例，**最多2句话**，然后立即回到主线
- 重点是"转折"或"新角度"，不是"新故事"

## 第二部分：黄金写作结构（严格按字数分配！）

```
【金句引用】30-50字
    ↓
【代入共鸣】60-80字 ← 用"你"代入读者（精简！）
    ↓
【强对比1】60-80字 ← "同样是XX，有的人...有的人..."
    ↓
【感慨/俗语】40-60字 ← 穿插老话、成语
    ↓
【强对比2或深入】60-80字
    ↓
【转折升华】40-60字 ← "但话说回来..."
    ↓
【希望/出路】40-60字 ← 正向引导
    ↓
【金句收尾+互动】30-40字
```

**总字数**：**450-550字**（不要超过600字！）

### ⚠️ 字数控制技巧
1. **写到400字时**：开始准备收尾
2. **写到500字时**：必须进入结尾段
3. **写到550字时**：立即结束
4. **每个段落不要超过80字**
5. **宁可精简，不要啰嗦**

**⚠️ 核心原则**：
- **用"你"贯穿全文**，让读者觉得"说的就是我"
- **用对比制造冲击**，而非讲长故事
- **故事只是引子**，如果用，最多2-3句
- **结尾要有希望**，不能全是负面
- **宁短勿长**，精简比啰嗦好

## 第三部分：两种排版风格（任选其一）

### 风格A：短句换行型（适合深度剖析类）

**特点**：每句话单独一行，像诗歌节奏

**案例片段**：
```
他们整天活在自己的小世界里。

用自己的旧知识解释一切。

从不怀疑自己可能是错的。

喜欢抬杠，不承认无知。

最关键的是，从不反思。
```

**关键特征**：
- 每句5-20字，单独成行
- 大量空行分隔意群
- 可以用"第一、第二"分段，但小标题要有态度
- 穿插反问："为什么？""不是吗？"

---

### 风格B：长段落型（适合感悟说理类）

**特点**：超长逗号句，一气呵成

**案例片段**：
```
很多人为养老焦虑，也为后边几年的养老做足了准备，钱得有，那是底气，孩子得有，那是依靠，伴侣最好也要有，那是拐杖，可当你真的老了，病了，就不是那么回事了。
```

**关键特征**：
- 超长逗号句（50-80字，只用逗号）
- 口语化："有点儿""的确是""所以啊"
- 设问自答
- 俗语穿插

## 第四部分：降低AI率核心技巧

### 技巧1："的"字结构（每篇10处+）★★★★★

**必用词组**：
- 租的房子、骑的车、打的工、受的罪
- 错过的人、走散的缘、花的钱、吃的苦
- 发的工资、上的学、看的病、说的话

**新增**：必须在故事段落中使用
- 他租的房子、他开的车、他打的工
- 我见过的人、我听过的话、我吃过的亏

---

### 技巧2：成语/俗语密度（每篇5个+）★★★★★

**高频成语池**：
- 失之东隅，收之桑榆
- 落子无悔，愿赌服输
- 汲汲营营、耿耿于怀、得不偿失
- 夏虫不可语冰，井蛙不可语海

**俗语池**：
- 痴心父母古来多，孝顺儿孙谁见了
- 人善被人欺，马善被人骑

**新增**：在反转段落后使用，加强说服力
```
【反转内容】
老话说得好，【成语/俗语】。
```

---

### 技巧3：双重否定（每篇2-3处）★★★★

**必用句式**：
- 人这一辈子，没有谁不栽跟头的
- 没什么大不了的
- 不会无缘无故的吃亏
- 不是说子女不孝顺，只是...

---

### 技巧4：分号对仗句（每篇1处）★★★★

**示例**：
- 繁华三千，看淡即是浮云；烦恼无数，想开就是晴天。
- 钱没了，可以再赚；情散了，可以再遇。

---

### 技巧5：叠词+儿化音（每篇4处+）★★★

**叠词**：干干净净、舒舒服服、普普通通、平平淡淡

**儿化音**：有点儿、烦心事儿、没事儿、那会儿

---

### 技巧6：比喻/意象（每篇2-3个）★★★★

**精彩比喻参考**：
- "像块茅坑里的石头，又臭又硬"
- "就像一个水杯，已经满了"
- "烂掉的关系，就像缠住脚的藤蔓"

---

## 第五部分：完读率关键技巧

### 技巧1：具体化数字（每篇3处+）★★★★★

数字比形容词有10倍冲击力！

**必改**：
- ❌ 很贵的房子 → ✅ 500万的江景房
- ❌ 收入差距大 → ✅ 月薪3千vs月薪3万
- ❌ 年纪大了 → ✅ 45岁、奔五的年纪
- ❌ 孩子小 → ✅ 3岁的娃、上小学二年级

---

### 技巧2：口语化追问（每篇2-3处）★★★★

制造对话感，拉近距离

**句式**：
- 你说，这能一样吗？
- 为什么？
- 你猜怎么着？
- 结果呢？
- 不是吗？

**禁止**：连续两句都用追问（会显得刻意）

---

### 技巧3：强互动结尾（必须执行）★★★★★

陈述句结尾完读率高但评论率低，影响二次推荐

#### 互动公式：
```
【总结金句】
    ↓
【空一行】
    ↓
**【提问/邀请评论】**
```

**示例**：
- 你身边有没有被钱逼到绝境的人？评论区说说。
- 你觉得这话对吗？来评论区聊聊。
- 你遇到过这种人吗？说出你的故事。

**关键**：
- 用"评论区"三个字（明确指令）
- 问题要具体，不要太宽泛

## 第六部分：禁忌清单

### ❌ 故事太长（最常见错误！）★★★★★

**这是生成内容最容易犯的错误**：

- ❌ "我一个朋友...他租的房子在六楼...他开的车是旧面包车...那天他从22楼走下来..."
- ❌ 花200字讲一个朋友的具体经历
- ❌ 故事细节太多：具体楼层、具体金额、具体场景

**正确做法**：
- ✅ 故事最多2-3句，然后立即转到"你"
- ✅ 用"你"代入读者，而非讲别人的故事
- ✅ 对标作品大多用"你是不是也这样？"开头

### ❌ 表达模板化（容易暴露AI）★★★★★

**每篇都用同样的句式，会让读者审美疲劳**：

- ❌ 每篇都用"这话，像一把冰冷的尺子..."
- ❌ 每篇都用"这话，像一面冰冷的镜子..."
- ❌ 每篇开头都是"一段很有道理的话："
- ❌ 每篇都用"你说，这能一样吗？"（可以用，但不要每段都用）

**正确做法**：
- ✅ 开头引用格式要变化（有人名用人名，没有就随机选）
- ✅ 过渡句要多样化，不要每篇都用同一个比喻
- ✅ "同样是XX"的对比不要超过2组
- ✅ 同一篇文章内的句式也要变化

### ❌ 完读杀手句式（绝对禁止）

以下句式会让人瞬间弃读：

- 总之、综上所述、总而言之（暗示"要结束了"，读者会跳过）
- 我们应该、让我们一起（说教感）
- 在当今社会、随着时代发展（八股文感）
- 不仅...而且...、之所以...是因为...（AI感太强）
- 值得注意的是...（教科书感）

### ❌ 慎用句式（每篇最多1次）

- 第一、第二、第三（超过3点会疲劳）
- XX说过、有句话说得好（用多了像鸡汤）
- 我一个朋友/我邻居（只能用1次，且最多2-3句）

### ❌ 其他禁忌

- 句子长度太均匀（要长短交替）
- 段落结构太规整（要随机）
- 排比太整齐（要故意不对称）
- 结尾太"金句化"（要自然收束+互动）
- **全文都在讲一个人的故事**（应该用"你"代入读者）

## 第八部分：对比案例

### ❌ 低完读率版本（平铺直叙）

```
一段很有道理的话："人控制钱，是生活；钱控制人，是生存。"

这话说得很对。

有钱人和没钱人，过的是不同的生活。

【直接讲道理】

【分点论述】

【金句结尾】
```

**问题**：无故事、无悬念、无互动

---

### ✅ 高完读率版本（故事+悬念+互动）

```
一段很有道理的话：
"人控制钱，是生活；钱控制人，是生存。"

我一个朋友，去年还在感慨自己过得憋屈。
今年突然顿悟了。

他说，以前总觉得有钱人也不过如此，谁还不是两只眼睛一张嘴，谁还不是吃饭睡觉上厕所。

直到有一次公司聚餐。

饭桌上，老板随口说："最近压力大，晚上睡不着。"
他心想，可不是嘛，我也失眠，愁的是房租够不够，孩子奶粉钱有没有着落。

结果老板接着说："想着要不要把那套江景房出了，换个大点儿的。"

他当时就愣住了。

同样是失眠，一个愁的是500万的房子换不换，一个愁的是下个月3000块的房租交不交。你说，这能一样吗？

老话说得好，人比人得死，货比货得扔。

不是说有钱就一定幸福，但没钱，连选择痛苦的权利都没有。你只能在那个逼仄的空间里，日复一日地熬着。

前两天看到个新闻，一个开厂的老板跳楼了。欠了几百万，实在扛不住了。

你看，有钱人的崩溃，是从几百万开始的；没钱人的崩溃，可能就是一场感冒、一次失业。

所以啊，别羡慕谁，也别可怜谁。把自己的日子过好，手里有点儿余粮，心里才能踏实。

钱这东西，够用就行，但不能没有。

你身边有没有被钱逼到绝境的人？评论区说说。
```

## 第九部分：敏感词过滤与合规机制 ★★★★★

### 重要性说明

即使内容再优质，触发平台违规会导致：
- 不予推荐（直接0曝光）
- 限流处罚（仅推荐给少量粉丝）
- 扣信用分（影响账号权重）
- 严重者禁言/封号

**原则**：宁可少赚流量，不可触碰红线

---

### 一、政治敏感类（零容忍）

#### ❌ 绝对禁止
- 任何国家领导人姓名、职务、政治事件
- 社会负面事件（暴乱、群体事件）
- 民族、宗教矛盾、地域歧视
- 质疑主流价值观、攻击社会制度

### 十、高危选题特殊注意

#### 贫富对比类（如"有钱人vs没钱人"）

**必须遵守**：
1. **绝不煽动对立** - 不批判富人，不煽动仇富
2. **保持客观中立** - 可感慨现实，但要有"向上出口"
3. **结尾必须正向** - "努力让自己从被钱控制到控制钱"
4. **避免极端案例** - 用"有的人""有些情况"泛指

#### 常见高危词替换
```
❌ "被钱逼到绝境的人" → ✅ "为钱发愁的时候"

❌ "苟活着" → ✅ "勉强过日子""将就着活"
❌ "没有希望" → ✅ "日子虽难，但还要往前走"
```
