# 3.2 运营管理功能


## F-MGT-01 用户管理

### 需求描述
管理员用于维护平台使用人员信息的后台功能。

### 功能要点
1. **基础维护**：支持单条添加、删除、禁用和重置密码。
2. **批量导入**：**支持通过 Excel 标准模板批量上传用户信息**，极大提升系统初始化效率。
3. **状态追踪**：记录用户最后在线时间与活跃状态。

---

## F-MGT-02 角色管理 (RBAC)

### 需求描述
基于角色的访问控制模型，实现灵活的权限分配。

### 功能要点
1. **角色定义**：支持创建自定义角色名称与描述（如：业务分析师、数据开发、管理员）。
2. **功能授权**：勾选功能模块权限。
3. **范围控制**：初步定义角色对数据的可见范围。

---

## F-MGT-03 场景知识库管理

### 需求描述
系统管理的核心单元，用于定义一个具体的业务查询域。

### 功能要点
1. **连接配置**：维护该场景对应的底层数据库连接（IP, 端口, 账号等）。
2. **语义挂载**：关联具体的 WrenAI MDL 语义物理模型。
3. **场景描述**：定义该场景的业务边界，帮助 Agent 更好地理解用户意图。

---

## F-MGT-04 用户与场景授权

### 需求描述
建立用户、角色与场景知识库之间的多对多映射关系。

### 功能要点
1. **快速绑定**：支持按用户或按角色进行批量授权。
2. **可见性控制**：确保用户在 Landing Page 只能看到其被授权的场景。

---

## F-MGT-05 大模型 API 接入管理

### 需求描述
灵犀问数引擎的“大脑”驱动源配置。

### 功能要点
1. **模型适配**：支持主流大模型（DeepSeek, GPT, Claude）及企业内部私有化部署模型的 API 配置。
2. **动态调整**：可配置 Token 限制、Temperature 参数及默认 Prompt。
3. **服务状态**：实时监控 API 响应状态。

---

## F-MGT-06 多租户管理 (进阶)

### 需求描述
(进阶预留) 支持企业内不同部门或子公司作为独立租户进行管理。

### 功能要点
1. **逻辑隔离**：租户间数据与配置物理/逻辑隔离。
2. **超级管理**：平台级管理员具备全局管理视角。
